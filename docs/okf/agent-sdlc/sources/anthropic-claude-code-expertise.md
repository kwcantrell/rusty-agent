# Summary

First-party material by Anthropic (published 2026-06-16). Key claims extracted below; the live document is the authority.

# Key claims

- AI coding agents handle implementation decisions while humans retain planning authority
- Domain expertise amplifies agent productivity per instruction across implementation tasks
- Agents execute complex multi-step workflows autonomously between human checkpoints
- Success verification relies on observable artifacts from test execution and version control
- Sessions encountering failures show differential recovery rates based on human expertise
- Operational tasks like deploying, configuring, and monitoring comprise significant agent workload
- Task complexity and economic value increased as agent capabilities improved over time
- Success measurement frameworks track multiple outcome types throughout sessions
- Workplace integration reveals shifting task composition toward higher-value work over time

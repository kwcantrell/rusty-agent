# Summary

First-party material by Academic research (published 2025-07-20). Key claims extracted below; the live document is the authority.

# Key claims

- Large-scale dataset of agent operations in real production settings, capturing 456,000 pull requests from five leading autonomous coding agents across 61,000 repositories and 47,000 developers.
- Performance-utility gap: autonomous coding agents achieve faster execution and higher code submission volume than humans, but their pull requests are accepted less frequently, indicating trust and utility concerns despite speed advantages.
- Agent-generated code exhibits lower structural complexity as measured by standard code complexity metrics compared to human-authored code.
- Autonomous coding agents operate across multiple phases of the development lifecycle, actively initiating, reviewing, and evolving code at scale in real repositories.
- Agents dramatically accelerate pull request submission rates within development workflows, with documented cases of developers submitting as many PRs in three days through agent assistance as they had in three years previously.
- Real-world evaluation data enables assessment of agent performance beyond synthetic benchmarks, addressing limitations of benchmark-only evaluation approaches.
- Multiple distinct autonomous coding agents (spanning OpenAI Codex, Devin, GitHub Copilot, Cursor, and Claude Code) are simultaneously active in production SDLC environments, creating a multi-agent ecosystem.

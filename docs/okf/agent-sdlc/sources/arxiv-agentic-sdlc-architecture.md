# Summary

First-party material by Anthropic, OpenAI, Google DeepMind, Microsoft Research, Princeton, Stanford (published 2026-04-29). Key claims extracted below; the live document is the authority.

# Key claims

- A formal six-layer reference architecture has been defined to structure how agentic software engineering systems operate.
- Operational scope of AI tools has shifted from line/function granularity to repository, feature, or algorithm granularity.
- An emerging Agentic SDLC (A-SDLC) framework has been defined, contrasting with traditional SDLC practices.
- The central focus in agent-driven development has shifted from pure code generation to delegated execution under human supervision.
- Benchmark performance on SWE-bench Verified has risen dramatically from 1.96% to 78.4% between October 2023 and April 2026.
- Controlled studies in production environments document time savings ranging from 13.6% to 55.8% when using AI agents.
- By 2026, approximately 49% of jobs sampled by Anthropic had integrated AI agents for at least a quarter of their tasks.
- Five critical open problems have been identified that will determine the success of agentic transition: evaluation, governance, technical debt, skill redistribution, and the economics of attention.

# Key claims

- LangGraph provides core capabilities for agent development including persistence, streaming, debugging support, and deployment readiness.
- Multiple workflow and agent patterns exist in LangGraph, each suited to different development scenarios.
- Orchestrator-worker patterns enable dynamic, unbounded subtask generation during agent execution.
- Evaluator-optimizer workflows support iterative refinement cycles, where feedback drives response recreation.
- LangSmith enables comparative testing and tracing of different workflow patterns for evaluation.
- LangSmith Engine provides automated monitoring of deployed systems, detecting issues and proposing fixes.

# Summary

Tauri's extensibility is showcased through three main categories. The "Official Features" section lists 34 built-in plugins covering functionality like file system access, notifications, HTTP clients, and biometric authentication. The "Community Plugins" section highlights over 60 developer-created extensions, including analytics tools, Bluetooth support, and specialized device integrations. Additionally, "Community Integrations" provides frameworks and adapters for popular tools like Deno, Angular, and Svelte.

A compatibility matrix displays which of the official plugins support Android, iOS, Linux, macOS, and Windows platforms. Tauri comes with extensibility in mind, enabling developers to extend their applications with both official and community-maintained resources.

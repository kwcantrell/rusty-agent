# Summary

"In order to get started building your project with Tauri you'll first need to install a few dependencies."

**Installation Categories:**

1. **System Dependencies** – Vary by operating system (Linux distributions, macOS Catalina 10.15+, Windows 7+)

2. **Rust** – Core language for Tauri development via `rustup`. Instruction: "For full support for Tauri and tools like trunk make sure the MSVC Rust toolchain is the selected default host triple."

3. **Node.js** – Required only for JavaScript frontend frameworks: "Only if you intend to use a JavaScript frontend framework"

4. **Mobile Configuration** (optional) – Android Studio and iOS tools needed for mobile app targeting

**Post-Installation:** Developers should "restart your Terminal" after installations to ensure changes take effect. Troubleshooting guide available for installation issues.
